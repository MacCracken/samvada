# Proposals

Pre-ADR design drafts. A proposal explores a decision in enough
depth that the eventual [ADR](../adr/) is a short ratification of
the chosen option rather than the place the analysis happens.

A proposal is **not** a decision — it scopes the options, the
trade-offs, the cost, and the open questions. When the decision
is made, it lands as an ADR and the proposal is linked from it.

**Never renumber** — proposals keep their number for life, like
ADRs.
